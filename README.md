# website

Public site for [Ariadne](https://github.com/Ariadne-Dev) — **ariadne.pablovallejo.dev**

Built with [Astro](https://astro.build), deployed on [Vercel](https://vercel.com).

## Develop

```bash
pnpm install
pnpm dev
```

Open [localhost:4321](http://localhost:4321).

## Deploy (Vercel)

1. Import `Ariadne-Dev/website` in Vercel
2. Framework: Astro (auto-detected)
3. Add domain `ariadne.pablovallejo.dev`

No environment variables required for the static site.

## Structure

```
src/
  layouts/Layout.astro   — page shell
  pages/index.astro    — home
  styles/global.css    — typography and colors
public/
  favicon.svg
```
